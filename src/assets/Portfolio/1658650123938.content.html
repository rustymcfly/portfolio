Einleitung

Markführer brauchen Tools für Kunden, die die Onlinepräsenz von Mitbewerbern abheben. Nicht nur die Technische Umsetzung ist hierbei von Bedeutung, sondern auch die Planung des Workflows und die Kommunikation mit dem Auftraggebenden

Deshalb konnte ich bisher eine Vielzahl von Erweiterungen für Shopware 5 und Shopware 6 erstellen.

Projekte

  • Für einen Namhaften Großhändler in Sachen Anlagentechnik konnte ich mehrere Plugins als "Toolbox" für dessen Kunden erstellen.
    • Ein Ersatzteilbeschaffungstool
      Dies beinhaltete die eine Renderingengine auf Canvasbasis, die zuvor im Backend mit der selben Engine erstellte Positionsreferenzen dem Kunden ausgibt. Eine völlig Produktunabhängige Konfiguration die noch in der Runtime dann mit den jeweiligen Ersatzteildaten aus Freitextfeldern/Custom Fields befüllt wird
    • Ein Tool zur Prüfung von Materialbeständigkeit
      Anhand Produktmerkmalen und einer definierten Matrix wird dem Kunden schnell und einfach die Beständigkeit gegen verschiedenste Medien offengelegt
  • Für einen Sanitärvertrieb
    • Artikelimport via Datanorm
      Mit wenig Aufwand kann der Kunde Datanorm zu Shopware importieren
    • Automatisierter Preisabgleich via FTP-Verbindung zum Großhändler
      Per einfache Kalkulationsformeln und Regeln werden Preise Tagesaktuell an den Markt angepasst
    • Automatischer EnEv-Label import per Verbindung zum Großhänler
      Ohne großen Verwaltungsaufwand besteht nun Compliance zur aktuellen EnEV
